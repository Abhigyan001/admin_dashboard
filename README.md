# Admin Dashboard
Admin Dashboard shows various statistics of data that a modern app dashboard contains. This project follows a tutorial by Ed Goh and is an attempt to practice React, Material UI, Nivo Charts, Formik, Yup, FullCalendar, and Data Grid. This application consists of Light and Dark Mode, 4 different Charts, 3 different Data Table Pages, FAQ Page, Form Page, and Calendar Integration.

## Getting Started

### Prerequisites

To get this project up and running locally, you must already have npm or yarn installed on your computer

**To get this project set up on your local machine, follow these simple steps:**

1. Open Terminal.
2. Navigate to your desired location to download the contents of this repository.
3. Copy and paste the following code into the Terminal: git clone https://github.com/Abhigyan001/admin_dashboard.git
4. Run ```cd admin_dashboard```.
5. Run ```npm install``` to get the necesary gems.
7. Run `npm start`
